# powerbi-learning-
Topics Covered: Data Visualization Concepts
Key Areas: Bar Charts, Pie Charts, Tables, and Filters vs Slicers
Today I spent some time exploring the basics of data visualization and how different visuals help present data in a clear and meaningful way. I looked at visuals such as bar charts, pie charts, and tables, and understood how each one helps highlight patterns, comparisons, or summaries within a dataset.
I also learned about the difference between filters and slicers in reports. Filters work in the background to limit or refine the data shown in a report or visual, while slicers act as interactive controls that allow users to easily filter and explore the data directly within the report interface.
